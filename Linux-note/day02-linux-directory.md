Linux基本目录结构

```
home: 创建用户目录
bin: 常用指令 
etc: 配置文件\环境目录 相关文件
boot: 启动linux相关文件
dev: 设备管理器
```

* 具体目录结构

```
目录1：
/bin [常用]  （/usr/bin、/usr/local/bin ）
- 这个目录存放着最经常使用的命令

/sbin (/usr/sbin、/usr/local/sbin)
- s: super User 这里存放系统管理员使用的系统管理程序

/home [常用] 
- 存放普通用户的主目录，在Linux中每一个用户都有自己的目录，一般该目录名是以用户的账号命名

/root [常用]
- 该目录为系统管理员，超级权限者的用户主目录

/lib 
- 系统开机所需要最基本动态连接共享库,其作用类似于windows里的DLL文件，几乎所有的程序都需要用到这些共享库

/lost+found 
- 这个目录一般为空，当系统非法关机，回存放一些数据

```

```
目录2:
/etc [常用]
- 所有的系统管理所需要的配置文件和子目录 程序服务配置、系统配置 my.conf

/usr [常用]
- 这是一个非常重要的目录，用户的很多应用程序和文件都放在这个目录下
类似于ndows的program files 目录

/boot 常用
- 存放启动Linux时使用的一些核心文件，包括一些连接文件以及镜像文件

/proc 
- 这个目录是虚拟的目录,它是系统内存的映射，访问这个目录来获取系统信息

/srv 
- service缩写，该目录存放一些服务启动之后所需要提取的数据

/sys
- 这是Linux2.6内核的一个很大的变化，该目录下安装了2.6内核中新出现的一个文件系统 sysfs

/tmp
- 这个目录是用来存放一些临时文件
```

```
目录3:
/dev
- 类似于设备管理器，把所有的硬件用文件的形式存储

/media [常用]
- Linux系统会自动识别一些设备，例如u盘、光驱等，当识别后，Linux会把别的设备挂载到这个目录下

/mnt [常用]
- 系统提供该目录是为了让用户临时挂载别的文件系统的，我们可以将外部的存储挂载在/mnt/上，
然后进入该目录就可以查看里面的内容。 共享目录/myshare

/opt
- 这是给主机额外安装软件所摆放的迷路。例如安装ORACLE数据库就可放到该目录下。默认为空（文件） 

/usr/local [常用]
- 这是另外一个给主机额外安装软件所安装的目录。一般是通过编译源码方式安装的程序(安装目标目录)

/var [常用]
- 这个目录中存放着在不断扩充着的东西，习惯将经常被修改的目录放在该目录下。
包括各种日志文件

/selinux [security-enhanced linux]
SELinux 是一中安全子系统，它能控制程序只能访问特定文件，有三种工作模式，可以自行设置
```
